# Propósito

Este repositório reúne todo o material utilizado na apresentação **Redes Neurais na Prática**, desenvolvida para a disciplina de **Matemática Discreta**.

A apresentação está dividida em dois encontros de aproximadamente **80 minutos** cada:

- **Aula teórica:** introduz os principais conceitos de redes neurais artificiais, incluindo a estrutura de um neurônio, funções de ativação, função de perda, gradiente descendente, retropropagação (*backpropagation*) e treinamento de modelos.
- **Aula prática:** aplica os conceitos apresentados anteriormente por meio de um notebook interativo em Python, no qual uma rede neural é construída e treinada passo a passo, permitindo visualizar como cada componente estudado atua durante o processo de aprendizagem.

O repositório contém:
- **Slides** utilizados na aula teórica;
- **Notebook prático** com implementações e experimentos realizados durante a aula;
- Recursos auxiliares utilizados para complementar a apresentação.
